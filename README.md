## Pry.js
An Interactive Javascript REPL

### Package installation

```
npm i -g @natasha08/pry.js
```

### Usage

Open a new REPL Session

```
pryjs
```

Help options

```
pryjs --help
```

#### REPL Commands

Clear Session data

```
.clear
```

Exit REPL

```
.exit
```

Exit REPL while saving session data

```
.saveexit
```
